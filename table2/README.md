# 📚 Collection de Projets TypeScript

Cette collection contient **13 projets TypeScript complets** couvrant tous les aspects essentiels du langage.

## 🚀 Comment utiliser

### Compilation et exécution d'un projet :

```bash
cd "nom-du-projet"
tsc                    # Compiler
node dist/index.js     # Exécuter
```

### Test automatique de tous les projets :
